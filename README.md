# 🤖 BhavdeepAI — Interactive AI Portfolio

An AI-powered developer portfolio with a terminal-style UI.

## 🚀 Features
- AI assistant (Gemini API)
- Secure backend (API key hidden)
- Command-based + natural language interaction
- Terminal-style UI
- Full-stack architecture (React + Node.js)

## 🛠 Tech Stack
- Frontend: React, Vite
- Backend: Node.js, Express
- AI: Google Gemini API
- DevOps: Docker, AWS (projects)

## 📦 Setup

### 1. Clone repo
git clone https://github.com/bhavdeepsingh7/bhavdeep-ai

### 2. Install
cd client && npm install  
cd ../server && npm install  

### 3. Add env
client/.env → VITE_API_URL=http://localhost:3000  
server/.env → GEMINI_API_KEY=your_key  

### 4. Run
npm run dev

Built to showcase full-stack + ML skills for internships.
